Library Management System

A simple and efficient way to manage your library resources

Welcome to the Library Management System

This is a simple library management system built with Ruby on Rails.

Note: This is a static page as the application is currently running without database access.

Books

Manage the library's book collection, including adding new books, updating information, and tracking availability.

Members

Manage library members, including registration, membership details, and borrowing history.

Loans

Track book loans, returns, and manage overdue items efficiently.

Features

  • Book inventory management
  • Member registration and management
  • Loan tracking and management
  • Fine calculation for overdue books
  • Search functionality for books and members
  • Reporting and analytics
